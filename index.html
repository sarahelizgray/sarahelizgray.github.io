Step 1: Install git

Get a copy of git from http://git-scm.com/downloads. During installation, select the option "Run Git and included Unix tools from the Windows Command Prompt." This will allow you to use the git help command, which uses the Unix man tool.

After installation, open a terminal window (on Windows, click the Start menu and type cmd). Enter the command git in a newly opened terminal/command prompt window. If the installation succeeded, this will show you a list of git commands.

Step 2: Setup GitHub

    Create a GitHub account at https://github.com/signup/free
  • On your local machine, make a directory called git. This directory will enclose all of your git repos for easy linking between your local machine and GitHub.
  • cd into the git directory
  • follow the "setting up git" portion of instructions at https://help.github.com/articles/set-up-git
  • now run git init to link your local machine to Github

Step 3: Fork the reptiles repo and complete class exercises

Fork the reptile repo at https://github.com/sarahelizgray/reptiles. Once you have forked the repo to your GitHub account, get it on your local machine by cloning.

git clone git@github.com:/reptiles.git

Now, test that your local and remote repos are talking to each other.

git pull

You should see "Already up-to-date"

git push

You should see "Everything up-to-date"
*note -- when pushing, you will always be asked for your github username and password when pushing. You can omit this step by completing the rest of the setup git instructions from github.

Exercises for this step are in the repo's README file.

Step 2: Install Ruby

To install githug on your own computer, first install Ruby. Ruby is a very nice programming language, similar to Python. If you are using Mac OS X, you already have Ruby. There are various ways to install Ruby on a Windows computer; I used http://rubyinstaller.org/; when you run the installer, choose the option to add the Ruby executables to your PATH.

Step 4: Install githug

Open a terminal window (on Windows, click the Start menu and type cmd), then enter the command

    gem install githug

Step 5: Play githug

From the command line, type githug, and start playing. This game covers concepts beyond the scope of this class, you will need the man git/git help and google to do some research. Have fun!
